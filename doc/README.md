# Valkrypt - Documentacio del projecte

Aquest directori conte la documentacio de lliurament del projecte Valkrypt.

## Documents principals

Els documents obligatoris que han d'estar directament dins de `/doc` son:

| Document | Estat | Fitxer |
| --- | --- | --- |
| Presentacio comercial / funcional | Fet | `comercial_2425_Valkrypt.pdf` |
| Presentacio tecnica | Fet | `tecnica_2425_Valkrypt.pdf` |
| Pitch publicitari | Fet | `pitch_2425_Valkrypt.mp4` |
| Demo de l'aplicacio | Pendent de substituir | `demo_2425_Valkrypt.mp4` |
| Presentacio resum | Fet | `summary_2425_Valkrypt.pdf` |
| Disseny | Fet | `disseny_2425_Valkrypt.pdf` |

> Atencio: `demo_2425_Valkrypt.mp4` existeix, pero actualment pesa 0 bytes. Cal substituir-lo per un video real de demostracio abans del lliurament final.

## Organitzacio per apartats

### A. Resum de presentacio

Carpeta: `A_resum_presentacio/`

Inclou la presentacio resum del projecte i captures utilitzades per explicar visualment l'aplicacio.

### B. Planificacio

Carpeta: `B_planificacio/`

Inclou la planificacio del projecte, sprints, tasques i documentacio de validacio.

### C. Disseny

Carpeta: `C_disseny/`

Inclou mockups, wireframes i diagrames importants:

- Wireframes del flux de pantalles.
- Diagrama E/R.
- Diagrama UML de classes.
- PDF resum de disseny.

### D. Codi font

Carpeta: `D_codifont/`

Inclou la referencia al codi font, instal.lacio i desplegament. El codi de l'aplicacio es troba a `../Valkrypt-RPG/`.

### E. Documentacio tecnica

Carpeta: `E_documentacio_tecnica/`

Inclou l'explicacio tecnica del projecte, estructura general, components principals i punt d'entrada per a nous desenvolupadors.

### I. Manual d'usuari

Carpeta: `I_manual_usuari/`

Inclou el manual necessari perque un usuari pugui entendre i utilitzar l'aplicacio.
